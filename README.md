# simplemap - かんたん地図プロット

### 概要

Excelなどから住所一覧をコピペするだけで地図にマーカーが配置できます。

googlemap apiを利用しています。

### 使い方

タブ区切りの住所データテキストをコピーし、  データ取込Divのテキストエリアに貼り付けてプロット開始を押します

テキストの１行目は見出しである必要があります。
